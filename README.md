# mlp-svm-from-scratch
NumPy kullanılarak MLP ve SVM algoritmalarının sıfırdan implementasyonu ve optimizer karşılaştırmaları.


Projede:
- Multi-Layer Perceptron (MLP) modeli NumPy kullanılarak sıfırdan implemente edilmiştir
- Support Vector Machine (SVM) algoritması yüksek seviyeli ML kütüphaneleri kullanılmadan yazılmıştır
- SGD, Momentum, RMSProp ve Adam optimizasyon yöntemleri karşılaştırılmıştır
- HIGGS ve RCV1 veri setleri üzerinde deneyler yapılmıştır

## Kullanılan Teknolojiler
- Python
- NumPy
- SciPy
- Matplotlib
